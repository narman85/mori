OAuth Debug Info

Bu səhifəni production-da açıb Google OAuth login et və console log-larını yoxla:

  1. F12 açıb Console-a get
  2. Google OAuth ilə login ol
  3. Dashboard-a get
  4. Console log-larını screenshot et və paylaş

Axtardığımız məlumatlar:

  • 🔍 Dashboard: Current user info: {...}
  • User ID, email, name, username
  • isOAuth: true/false
  • fullUserObject

Bu məlumat əsasında OAuth user duplicate problemi həll edəcəm.
